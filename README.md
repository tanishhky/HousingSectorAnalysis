# Housing Sector Analysis

A data-driven exploration of regional real estate markets in India, utilizing extensive datasets to uncover trends, correlations, and pricing drivers.

![Correlation Analysis Plot](correlation_plot_numeric.png)

## Overview

Real estate markets vary drastically by region, and understanding these variances requires robust, localized data analysis. This repository houses an end-to-end data science study analyzing housing sectors across key Indian metropolitan areas (Gurgaon, Hyderabad, Kolkata, and Mumbai). 

## Contents
- `main.ipynb`: The core Jupyter Notebook encompassing data loading, cleaning, EDA (Exploratory Data Analysis), metric engineering, and statistical correlation tracking.
- `*.csv`: Extensive transactional and listings data for major cities.
- `correlation_plot_numeric.png`: Exported visualization capturing the mathematical relationships between various housing features like square footage, location index, and final price.
- `facets/`: Contains interactive or subset visualizations.

## Usage
Simply launch the `main.ipynb` notebook in an environment equipped with Pandas, Matplotlib, and Seaborn to reproduce the analysis.
